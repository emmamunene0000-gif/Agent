## EMOJI DICTIONARY (AGENT LANGUAGE — EXACT)

| Symbol | Meaning |
|--------|---------|
| 🟢 | BULL — bullish state confirmed |
| 🔴 | BEAR — bearish state confirmed |
| 🟡 | NEUT — neutral, slight bull lean |
| 🟠 | NEUT — neutral, slight bear lean |
| ⚪ | NEUTRAL — no directional edge |
| 🔥 | FULLY ALIGNED — 4-layer sovereign sync |
| ⚡ | TARGET — a specific price level |
| ○ | READY — signal is live, gate passed |
| ⊗ | WAIT — signal fired but gate not passed |
| ⚠️ | CT — counter-trend signal flagged |
| 🚫 | SKIP — signal explicitly rejected |
| 💧 | LIQUIDITY — liquidity level or sweep |
| 🧲 | NEAR — price approaching key level |

**Field names — use EXACT agent language:**
- V.WAP (not VWAP)
- Fib Trend (not Fibonacci)
- Regime (not RSI regime)
- ATM (not trailing stop)
- L1/L2/L3/L4 (not Sovereign/Anchor/Filter/Exec in short form)
- PDH / PDL (not Prev Day High)
- BSL / SSL (not Buy-side / Sell-side)
